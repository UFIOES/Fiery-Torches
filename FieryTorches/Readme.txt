Fiery Torches

By: UFIOES

To install, extract FieryTorches into TUG\Mods\
Then add "Mods/FieryTorches" to TUG\Config\mods.txt, it must be after "Game/Core".
e.g.

Mods
{
	"Game/Dev"
	"Game/Survival"
	"Game/Creative"
	"Game/Core"
	"Mods/UFI"
	"Mods/FieryTorches"
}
